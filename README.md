# 📊 Sales Performance Dashboard | Power BI + Python

## 🚀 Project Overview
This project focuses on analyzing retail sales data and building an interactive dashboard to generate meaningful business insights. It covers the complete workflow from **data cleaning and preprocessing to visualization and insights generation**.

---

## 🧹 Data Cleaning & Preparation
Data cleaning was performed using Python to ensure high-quality and reliable data for analysis.

### Key Steps:
- Handling missing values and duplicates  
- Converting data types (Date, Numeric)  
- Removing invalid records (negative quantity, cancellations)  
- Feature engineering (Year, Month, Revenue Tier)  
- Outlier detection and treatment  

---

## 📈 Dashboard Features (Power BI)
The Power BI dashboard is designed as a **single-page interactive report**.

### Includes:
- KPI Cards (Total Sales, Quantity, Orders)  
- Sales Trend Over Time (Line Chart)  
- Top Products by Revenue (Bar Chart)  
- Sales by Country/Region (Bar/Pie Chart)  
- Interactive Filters (Date, Country, Category)  

---

## 📊 Key Insights
- Top-performing products contribute most of the revenue  
- Certain regions generate higher sales than others  
- Sales show clear trends over time (monthly growth/decline)  
- Some products and regions underperform and need improvement  

---

## 🎯 Business Recommendation
To grow faster, the business should:
- Focus on high-performing products  
- Expand in profitable regions  
- Target peak sales periods  
- Improve or remove low-performing products  
- Retain high-value customers
